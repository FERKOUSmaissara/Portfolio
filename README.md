## Lancer le projet en local

Cloner le depot:

```bash
git clone https://github.com/FERKOUSmaissara/Portfolio.git
```

Entrer dans le dossier:

```bash
cd Portfolio
```

Ouvrir `index.html` dans un navigateur.

Ou utiliser un serveur local:

```bash
python -m http.server
```

Puis ouvrir:

`http://localhost:8000`
