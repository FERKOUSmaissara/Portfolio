# ⚙️ Installation locale

Ce projet est un **site statique**, il peut être lancé très facilement.

### 1️⃣ Cloner le dépôt


git clone https://github.com/FERKOUSmaissara/Portfolio.git


### 2️⃣ Accéder au dossier


cd Portfolio


### 3️⃣ Ouvrir le site

Ouvrir simplement :


index.html


dans votre navigateur.

---

# 🔧 Lancer avec un serveur local (option recommandé)

Si Python est installé :


python -m http.server


Puis ouvrir :


http://localhost:8000
